design notes & open questions

the bits worth deciding before turning this into Vue.

Behaviour

  • Nook is single-tap. Always navigates straight to `/nook`. The other three open a stack.
  • Tap-tab-to-toggle. Same tab tapped a second time dismisses the stack. So does scrim tap, back gesture, Escape.
  • Tap-bean-to-go. Each bean inside the stack navigates and closes the stack. Active route is ringed in Heritage Orange so users see where they're returning to.
  • One stack at a time. Tapping a different category swaps the stack instantly with a re-stagger (no full close + open).
  • Closed when scrolling. Vertical scroll on the page dismisses an open stack — keeps the chrome calm and avoids accidental hold-overs.

Spec mapping

  • Nook → `/nook` (homepage, today's view).
  • Planning 🌳 → activities, to-do, travel. Three items.
  • Money 🐷 → overview, accounts, budgets, transactions, goals, assets. Six items. Hidden when `canViewFinances` is off.
  • Pod 🌱 → meet the beans, scrapbook, cookbook, care & safety, emergency. Five items. Settings & help live in the existing top-right avatar menu, not in the bar.
  • FAB stays. Quick-add `+` keeps its corner; opening the FAB closes any open nav stack and vice versa.

Open questions

  • Visual cue for "has more". Mockup uses a tiny dot in the tab corner that grows + tints orange when the stack is open. Alternatives: a hairline underline, a chevron, or nothing (rely on the spring motion to teach).
  • What if `canViewFinances` is off? Bar collapses to 3 tabs (Nook / Planning / Pod). Grid stretches; spacing still works.
  • Active-route highlight. Tab shows the orange top-bar when the user is on any route inside its category — so a user on `/transactions` sees Money lit even when the stack is closed.
  • Reduced motion. All variants gracefully fall back to a fade-in (no spring stagger) when `prefers-reduced-motion: reduce` is set.
  • Naming. "Planning" is the working label for the treehouse minus pod. Could also be "Plan" (shorter), "Calendar" (narrower), or keep the tree emoji and label "Treehouse" (brand-true). Greg to call.
